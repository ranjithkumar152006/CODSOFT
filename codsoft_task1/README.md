## Rule-Based Chatbot using Python

# Description

This project implements a simple rule-based chatbot using Python.
The chatbot interacts with users through the command line and responds
to predefined inputs using conditional statements and keyword matching.

# Features

    - Responds to greetings such as hi and hello
    - Answers basic questions like name and how are you
    - Displays the current system time
    - Handles unknown inputs gracefully
    - Exits safely when the user types exit
    - Simple and beginner-friendly logic

# Technologies Used

    - Python 3.X
    - Visual Studio Code
    - Command Prompt (CMD)

# How to Run the Program

    - Make sure Python is installed on your system
    - Open the project folder in Visual Studio Code
    - Open the terminal or Command Prompt
    - Run the program using:
        python simple_chatbot.py
        Start interacting with the chatbot

# Sample Output

Bot: Hello! I am a simple chatbot. You can ask me a few questions.
You: hi
Bot: Hello! How can I help you today?
You: what is your name
Bot: I am ChatBot 1.0, your friendly assistant!

Bot: Goodbye! Have a nice day.
